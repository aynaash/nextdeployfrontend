# Installation

NextDeploy CLI is the primary tool for managing your deployments. This guide covers installation on all major operating systems and troubleshooting common issues.

## System Requirements

Before installing NextDeploy, ensure your system meets these requirements:

- **Node.js**: Version 16.0.0 or higher
- **npm**: Version 7.0.0 or higher (comes with Node.js)
- **Operating System**: Windows 10+, macOS 10.15+, or Linux (Ubuntu 18.04+, CentOS 7+)
- **Memory**: At least 2GB RAM available
- **Disk Space**: 500MB free space for CLI and dependencies

<Info>
We recommend using the latest LTS version of Node.js for the best experience and security updates.
</Info>

## Configuration

### Initial Setup

Run the initial setup to configure NextDeploy for your environment:

<CodeBlock title="Initial setup" language="bash">
{`# Run setup wizard
nextdeploy setup

# Or configure manually
nextdeploy config set user.name "Your Name"
nextdeploy config set user.email "your.email@example.com"`}
</CodeBlock>

### Authentication

Set up authentication for NextDeploy services:

<CodeBlock title="Authentication setup" language="bash">
{`# Login to NextDeploy (if using hosted services)
nextdeploy auth login

# Or set API token manually
nextdeploy config set auth.token "your-api-token"

# Verify authentication
nextdeploy auth whoami`}
</CodeBlock>

### Global Configuration

Configure global settings that apply to all projects:

<CodeBlock title="Global configuration" language="bash">
{`# Set default deployment strategy
nextdeploy config set defaults.strategy "blue-green"

# Set default monitoring
nextdeploy config set defaults.monitoring true

# Set default backup retention
nextdeploy config set defaults.backup.retention 5

# View all configuration
nextdeploy config list`}
</CodeBlock>

## Platform-Specific Setup

### Windows Setup

<CodeBlock title="Windows-specific setup" language="powershell">
{`# Install Windows Subsystem for Linux (recommended)
wsl --install

# Or install Git Bash for better terminal experience
# Download from: https://git-scm.com/download/win

# Add NextDeploy to Windows PATH
$env:PATH += ";C:\nextdeploy"
[Environment]::SetEnvironmentVariable("PATH", $env:PATH, "User")`}
</CodeBlock>

### macOS Setup

<CodeBlock title="macOS-specific setup" language="bash">
{`# Install Homebrew (if not already installed)
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# Install NextDeploy via Homebrew (alternative method)
brew tap nextdeploy/tap
brew install nextdeploy

# Update shell profile
echo 'export PATH="/usr/local/bin:$PATH"' >> ~/.zshrc
source ~/.zshrc`}
</CodeBlock>

### Linux Setup

<CodeBlock title="Linux-specific setup" language="bash">
{`# Ubuntu/Debian
sudo apt update
sudo apt install curl git

# CentOS/RHEL
sudo yum install curl git

# Add to shell profile
echo 'export PATH="/usr/local/bin:$PATH"' >> ~/.bashrc
source ~/.bashrc

# Set up completion (optional)
nextdeploy completion bash | sudo tee /etc/bash_completion.d/nextdeploy`}
</CodeBlock>

## Troubleshooting

### Common Issues

#### Command Not Found

<CodeBlock title="Fix command not found" language="bash">
{`# Check if NextDeploy is installed
npm list -g nextdeploy

# Check PATH
echo $PATH

# Reinstall if necessary
npm uninstall -g nextdeploy
npm install -g nextdeploy

# For npm permission issues on macOS/Linux
sudo chown -R $(whoami) $(npm config get prefix)/{lib/node_modules,bin,share}`}
</CodeBlock>

#### Permission Errors

<CodeBlock title="Fix permission errors" language="bash">
{`# Linux/macOS: Fix npm permissions
mkdir ~/.npm-global
npm config set prefix '~/.npm-global'
echo 'export PATH=~/.npm-global/bin:$PATH' >> ~/.bashrc
source ~/.bashrc

# Windows: Run as Administrator
# Right-click Command Prompt ‚Üí "Run as administrator"`}
</CodeBlock>

#### Version Conflicts

<CodeBlock title="Resolve version conflicts" language="bash">
{`# Check Node.js version
node --version

# Update Node.js if needed
# Visit: https://nodejs.org/

# Clear npm cache
npm cache clean --force

# Reinstall NextDeploy
npm uninstall -g nextdeploy
npm install -g nextdeploy@latest`}
</CodeBlock>

### Getting Help

If you're still experiencing issues:

<CodeBlock title="Diagnostic commands" language="bash">
{`# Run system diagnostics
nextdeploy doctor

# Check configuration
nextdeploy config list

# Enable debug mode
nextdeploy --debug <command>

# View logs
nextdeploy logs --system`}
</CodeBlock>

## Updating NextDeploy

Keep NextDeploy CLI up to date for the latest features and security fixes:

<CodeBlock title="Update NextDeploy" language="bash">
{`# Check current version
nextdeploy --version

# Check for updates
nextdeploy update check

# Update to latest version
nextdeploy update

# Or update via npm
npm update -g nextdeploy

# View changelog
nextdeploy changelog`}
</CodeBlock>

## Uninstallation

If you need to remove NextDeploy CLI:

<CodeBlock title="Uninstall NextDeploy" language="bash">
{`# Uninstall via npm
npm uninstall -g nextdeploy

# Remove configuration (optional)
rm -rf ~/.nextdeploy

# Remove shell completion
sudo rm /etc/bash_completion.d/nextdeploy

# Verify removal
which nextdeploy
# Should return: command not found`}
</CodeBlock>
